# Briefing do Projeto

## Tipo de Projeto
Aplicação Web / SaaS

## Objetivo de Negócio
Desenvolver uma mentora de vendas IA via WhatsApp que utiliza metodologias da Barbara D'Elia (BANT, SPIN Selling, Storytelling, Slow Pressure Selling) para capacitação e suporte contínuo de profissionais de vendas

## Público-Alvo
Vendedores (SDR, Closer, Account Executive) e empresários/gestores de vendas que buscam capacitação e suporte em metodologias de vendas

## Nicho de Mercado
Micro-SaaS de mentoria e treinamento em vendas via WhatsApp

## Escopo Desejado
Refatoração completa da Babi v3 - desenvolvimento do zero utilizando Windsurf, mantendo arquitetura com n8n (workflows já existem como templates), Supabase (algumas tabelas já criadas), com necessidade de documentação completa do projeto

## Stack Tecnológica
Frontend: React/Lovable com Supabase Auth, Backend: n8n (orquestrador), Supabase (PostgreSQL + pgvector), UAZAPI (WhatsApp Gateway), OpenAI GPT-4-mini/Google Gemini/Anthropic Claude (LLMs), ElevenLabs (áudio), Redis (cache)

## Prazo
3 dias (desenvolvimento do zero utilizando Windsurf com templates n8n existentes e tabelas Supabase já criadas)

## Orçamento
Apenas créditos das ferramentas: Windsurf, Claude e Lovable - sem orçamento monetário adicional

## Critérios de Sucesso
Plataforma funcional que oferece mentoria personalizada via WhatsApp, dashboard web para configurações, sistema de role play com áudio, e integração eficiente entre todos os componentes da arquitetura modular
